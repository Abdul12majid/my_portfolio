TaskMaster

TaskMaster is a dynamic task management system built with Django REST Framework, Celery, and Redis, designed to handle asynchronous task processing efficiently. The application enables users to create tasks, assign them, and trigger background notifications through an intuitive API.

Key Features:
  • Task creation, assignment, and management through a RESTful API.
  • AAsynchronous task execution with Celery for efficient background processing.
  • Scalable architecture using Redis for message queuing and task management.
